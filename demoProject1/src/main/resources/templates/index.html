    (*** : 프로젝트명)

    해당 클래스에는 main 메소드가 존재 -> 실행 -> 내장된 tomcat 서버가 배포를 시작
    (브라우저가 자동으로 열리지 않기 때문에 브라우저 주소창에 입력해서 접속해야 함)
