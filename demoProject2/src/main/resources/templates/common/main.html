2. Thymeleaf

	  웹 및 독립 실행형 환경을 모두 지원하는 최신 서버 측 Java 템플릿 엔진
	  -> 웹 실행    == 요청 시 포워드 되는 화면
	  -> 독립 실행  == html 파일만 실행(서버없이)
	  HTML 파일에서 th(Thymeleaf) 속성을 이용해
	  컨트롤러로 부터 전달 받은 데이터를 이용해 동적 페이지를 만들 수 있음.
	  Spring Boot에서는 JSP가 아닌 Thymeleaf 사용을 권장하고 있음.
